# Rush Hour Replica – Pixel Watch 4 Style (Wear OS)

A faithful replica of the Rush Hour watch face from Pixel Watch 4.  
Recreated with great attention to detail, including smooth animations and a fully optimized Always-On Display experience.

Built using the new Watch Face Format (WFF) for modern Wear OS devices.

---

## ✨ Features

- Clean and modern Pixel-style design
- Smooth animations
- Optimized Always-On Display (AOD)
- Lightweight and battery efficient
- Accurate time and system indicators

---

## ⚙️ Requirements

- Wear OS 4 or higher (WFF required)
- Compatible with modern Wear OS smartwatches
